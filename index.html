Problem Info-:


-> There is a Rat in a Maze
-> Maze Contain boxes having value 0 and 1
->The rat can only move horizontally
or 
vertically (not diagonally) 
and
can only move to boxes having value 1
-> Rat will start from box having value 'S'
and
Rat have to reach it's 
destination (box having value 'D')
-> select "source and then click on any box to mark it "S"
-> select "Destination and then click on any box to mark it "D"
-> select "1/0 and then click on any box to fill it with value 1/0"

-> After Filling all Values(D,S,0/1) , click on Start to Display 
a Valid Path For Rat to Move

Note-: if No source and destination is declared by the user(you) 
        By default value of source will start from -:
                upper left corner
        By default value of destination will start from -:
                lower right corner
